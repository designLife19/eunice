Overview

eResponse is a web application for helping vendors to submit electric form. e-BidForm is used for e-Biddingo purposes. When an e-BidForm is completed and the vid is submitted to the owner, they can import it directly into their compatible software. Using e-BidForm allows for electronic submission of bids.

I was involved as a UX / UI designer for the application. And This case study is about built a new UI.

Problem & Solving

Problem Statement

The main goal is redesigning eResponse web application UX / UI.
There are many bids and forms on the eResponse application and the number continues to grow. As a result, there is some feedback that there is some difficulty in finding and viewing data from the user's perspective. Users want to see a lot of data on one screen but want a simple layout that is not confusing.

Solving

Our team has decided to change the new UI, which requires the development of old UI for users.

1. Keep the interface simple and Utilizes the entire space for the user's convenience.
Designed to avoid unnecessary elements and make the bid detail information and data clearly visible.

View new UI for the List of eBidding

2. Created consistency and use common UI elements.
Using the "common elements" in the UI allows users to be more comfortable and get things done faster. To increase efficiency, created patterns with color, layout, and design throughout the application. It helps the user to see the data more faster and easier and to complete the work.

View new UI for Bid detail page

3. Build split sections
In terms of user requirements (an easy way to see all the data at a glance) and their perspective, our team chose split sections. This is much more effective in the following way:

  • Users have faster task-completion times
  • Users can easily define whether the content is right for them
  • Users make fewer errors in recollection
  • Users get a better sense of a pageâ€™s structure

View new UI for e-Bid Response

4. Created Collapsible panels.
It helps Users can only see the relevant content. Collapsible panels provide users with the ability to expand and collapse content as needed. They can simplify the interface by hiding content until it is needed. As a result, this is convenient for users in that they can find and check all data on one screen without confusion.
